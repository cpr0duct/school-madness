PROJECT BACKUPS FOLDER
======================

This folder stores backups of the entire RPG Maker MZ project.

Weekly_Backups/
- Create a backup every week
- Name format: SchoolMadness_YYYY-MM-DD.zip
- Keep last 4 weeks only (delete older ones to save space)

Milestone_Backups/
- Create a backup when reaching major milestones
- Examples: "v1.0_School_Bus_Complete.zip"
- Keep these permanently

HOW TO CREATE A BACKUP:
1. Close RPG Maker MZ
2. Go to your project folder on your computer
3. Right-click the project folder → Send to → Compressed (zipped) folder
4. Name it: SchoolMadness_YYYY-MM-DD.zip
5. Upload to this Google Drive folder

WHY BACKUP:
- GitHub is great for code, but backups provide extra safety
- Easy to restore if something breaks
- Can go back to previous versions
- Peace of mind!

SCHEDULE:
- Weekly backup: Every Sunday evening
- Milestone backup: After completing major features
