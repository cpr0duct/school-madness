SCHOOL MADNESS - GOOGLE DRIVE FOLDER STRUCTURE
==============================================

This is the shared Google Drive folder for the School Madness game development project.

INSTRUCTIONS:
1. Upload this entire "School Madness - Game Dev" folder to your Google Drive
2. Right-click the folder and click "Share"
3. Add all team members with "Editor" permissions
4. Share the link in Discord

FOLDER GUIDE:
- 01_Documentation/     - Game design docs, story, meeting notes
- 02_Art_Source_Files/  - Original art files (.psd, .aseprite, etc.)
- 03_Audio_Source/      - Music and sound effect source files
- 04_Project_Backups/   - Weekly backups of the RPG Maker project
- 05_Builds_and_Exports/ - Playtest builds and demos
- 06_Marketing_and_Promo/ - Screenshots, trailers, social media
- 07_Meeting_Recordings/ - Video recordings of meetings
- 08_Miscellaneous/     - Everything else

NOTE: GitHub stores the actual game project. This Drive is for:
- Large reference files
- Source art files (before export)
- Documentation and planning
- Backups and builds
