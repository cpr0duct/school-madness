DOCUMENTATION FOLDER
====================

This folder contains all project documentation.

SUBFOLDERS:

Design_Documents/
- Game Design Document (GDD)
- Technical specifications
- Asset tracking spreadsheets

Story_Writing/
- Story outline and plot
- Character profiles
- Dialogue scripts
- World building notes

Meeting_Notes/
- Create a new Google Doc for each meeting
- Name format: YYYY-MM-DD_Meeting_Topic.gdoc

Guides_and_Tutorials/
- RPG Maker MZ guides
- GitHub workflow guides
- Any helpful PDFs or resources
