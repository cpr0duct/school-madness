ART SOURCE FILES FOLDER
=======================

IMPORTANT: This folder stores ORIGINAL art files BEFORE export to RPG Maker.

Characters/
  Source_Files/ - .psd, .aseprite, .ai files with layers intact
  Reference/ - Inspiration images, concept sketches

Tilesets/
  Source_Files/ - Original tileset art before tile cutting
  Reference/ - Tileset inspiration and references

UI_Elements/
  Source_Files/ - Menu designs, buttons, HUD elements
  Mockups/ - UI design mockups and prototypes

Promotional/
  Logo_Designs/ - Game logo variations
  Title_Screen/ - Title screen artwork
  Marketing_Art/ - Promotional images

Reference_Images/
  School_References/ - Photos/images of schools, buses, etc.
  Character_References/ - Character design inspiration
  Color_Palettes/ - Color scheme references

WORKFLOW:
1. Create original art here in high resolution
2. Export to RPG Maker format
3. Exported files go in GitHub repository
4. Original files stay here for future editing
