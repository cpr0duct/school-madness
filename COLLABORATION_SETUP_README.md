# School Madness - Collaboration Setup
## Two-Family RPG Maker MZ Game Development Project

---

Welcome to **School Madness**! This is a collaborative RPG game project between two families:

**Team:**
- **Family 1** (Technical): Handles programming, RPG Maker events, and systems
- **Family 2** (Artists): Creates sprites, tilesets, and visual design
- **Kids from both families**: Ideas, playtesting, and collaboration

- [ ] **13. Hold Kickoff Meeting**
  - [ ] Join Discord voice channel
  - [ ] Introduce everyone
  - [ ] Review the tools together
  - [ ] Do a quick test:
    - [ ] Someone makes a small change in RPG Maker
    - [ ] Commits and pushes to GitHub
    - [ ] Everyone else pulls the change
    - [ ] Verify it works!

- [ ] **14. Plan First Week**
  - [ ] Fill out Game Design Document together
  - [ ] Create initial tasks in Trello
  - [ ] Assign first tasks to people
  - [ ] Set goals for first week

  - [ ] Pick weekly meeting day/time
  - [ ] Set up weekly backup reminder
  - [ ] Agree on communication expectations

## üìã Daily Workflow (Quick Reference)

### Before Working:
1. **Discord** ‚Üí Check #task-coordination (is anyone in RPG Maker?)
2. **Trello** ‚Üí Pick a task from "This Week"
3. **GitHub Desktop** ‚Üí Pull latest changes

### While Working:
4. Work on your task in RPG Maker or art software
5. Save frequently
6. Update Trello as you progress

### When Done:
7. Test your work
8. **GitHub Desktop** ‚Üí Commit and push changes
9. **Trello** ‚Üí Move card to "Done"
10. **Discord** ‚Üí Announce completion

**Full details:** See `docs/COMPLETE_WORKFLOW_GUIDE.md`
