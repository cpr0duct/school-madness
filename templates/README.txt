TEMPLATES FOLDER - School Madness RPG Maker MZ Project
======================================================

This folder contains template files to help Family 2 create tilesets for the game.

WHAT'S IN THIS FOLDER:
---------------------

1. template_SchoolBusInt_B_INSTRUCTIONS.txt
   - Instructions for creating the school bus interior tileset template
   - Includes exact dimensions and grid specifications

2. example_seat_layout.txt
   - ASCII art showing how to arrange seats on the tileset grid
   - Visual guide for tile placement

HOW TO USE THESE TEMPLATES:
---------------------------

1. Read the FAMILY2_SCHOOL_BUS_TILESET_GUIDE.md in the docs/ folder first
2. Follow the instructions in template_SchoolBusInt_B_INSTRUCTIONS.txt to create your canvas
3. Use your preferred art software (GIMP, Photoshop, Aseprite, etc.)
4. Create your tileset following the specifications
5. Export as PNG with the correct settings
6. Place the final PNG in: project/School Madness/img/tilesets/

IMPORTANT NOTES:
---------------

- All tilesets must be exactly the dimensions specified
- Use 8-bit indexed color PNG format
- Grid must be 48x48 pixels per tile
- Background must be fully transparent
- Name your files exactly as instructed (e.g., SchoolBusInt_B.png)
