TEMPLATE CREATION INSTRUCTIONS
School Bus Interior B Tileset Template
========================================

CANVAS SPECIFICATIONS:
--------------------

Dimensions: 768 x 768 pixels
Grid: 48 x 48 pixels per tile (16 tiles wide x 16 tiles tall)
Format: PNG
Color Mode: 8-bit indexed color (256 colors maximum)
Background: Fully transparent (alpha = 0)
Interlacing: Non-interlaced

STEP-BY-STEP TEMPLATE CREATION:
-------------------------------

OPTION A: Using GIMP (Free)
--------------------------
1. File → New
2. Width: 768 pixels
3. Height: 768 pixels
4. Advanced Options → Fill with: Transparency
5. Click OK
6. Image → Mode → Indexed
   - Select "Generate optimized palette"
   - Maximum colors: 256
7. View → Show Grid
8. Image → Configure Grid
   - Width: 48 pixels
   - Height: 48 pixels
   - Line style: Solid
9. View → Snap to Grid (enable this!)
10. Save as template_SchoolBusInt_B.png

OPTION B: Using Photoshop
------------------------
1. File → New
2. Width: 768 px
3. Height: 768 px
4. Background: Transparent
5. View → Show → Grid
6. Preferences → Guides, Grid & Slices
   - Gridline every: 48 pixels
   - Subdivisions: 1
7. View → Snap To → Grid
8. Image → Mode → Indexed Color
   - Palette: Adaptive
   - Colors: 256
9. Save for Web → PNG-8

OPTION C: Using Aseprite ($20)
-----------------------------
1. File → New Sprite
2. Width: 768 px
3. Height: 768 px
4. Color Mode: Indexed
5. Background: Transparent
6. View → Grid → Grid Settings
   - Width: 48 px
   - Height: 48 px
7. View → Grid → Show Grid
8. View → Snap to Grid
9. File → Export → PNG

WHAT TO DRAW IN THE TEMPLATE:
----------------------------

For a SIMPLE PLACEHOLDER school bus interior, create these tiles:

ROW 0-1: Bus Seats (Forward-Facing)
-----------------------------------
Columns 0-1: SEAT (Occupied - Top Half)
  - Draw seat back (green or blue vinyl)
  - 48px wide x 48px tall

Columns 0-1: SEAT (Occupied - Bottom Half)
  - Draw seat cushion
  - 48px wide x 48px tall

This creates a 48x96px (1x2 tile) bus seat.

Make 3-4 variations across the row:
- Empty seat
- Seat with simple silhouette of student
- Seat facing different direction
- Damaged/worn seat

ROW 2: Floor and Wall Tiles
---------------------------
Column 0: Floor tile (repeating pattern)
  - Simple gray or black rubber floor
  - Must tile seamlessly when repeated

Column 1: Wall panel
  - Metal bus interior wall
  - Light green or cream color

Column 2-3: Additional floor variations
  - Different wear patterns
  - Dirt or scuff marks

ROW 3-15: LEAVE EMPTY FOR NOW
-----------------------------
These rows are for future expansion:
- More seat variations
- Dashboard components
- Windows
- Decorative elements

COLOR PALETTE SUGGESTION:
-----------------------

Keep it simple for placeholder:
- Seat vinyl: #4A7C59 (dark green)
- Floor: #2F2F2F (dark gray)
- Wall panel: #C8D5B9 (light greenish)
- Outline/shadows: #1A1A1A (nearly black)
- Highlights: #FFFFFF (white, sparingly)

Total: ~5-10 colors maximum for simple placeholder

TESTING YOUR TEMPLATE:
---------------------

Before spending hours on art:
1. Create just ONE seat (2 tiles)
2. Create ONE floor tile
3. Export as PNG with correct settings
4. Import into RPG Maker MZ
5. Place on a test map
6. Verify:
   - Tiles align correctly
   - No gaps or overlaps
   - Colors look right
   - File imports without errors

If test works, continue with more tiles!

COMMON MISTAKES TO AVOID:
------------------------

❌ Canvas not exactly 768x768px
❌ Using 24-bit color instead of 8-bit indexed
❌ Tiles not aligned to 48px grid
❌ Semi-transparent pixels (use fully transparent or opaque only)
❌ Forgetting to enable "Snap to Grid"
❌ Saving as JPEG instead of PNG
❌ Making seat too detailed for placeholder (keep it simple!)

FINAL CHECKLIST:
